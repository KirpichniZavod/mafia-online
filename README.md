# Мафия Онлайн

Онлайн игра в мафию с веб-клиентом и сервером на Render.

## Стек технологий
- **Сервер:** Node.js + Express + Socket.IO + Prisma
- **База данных:** PostgreSQL
- **Клиент:** React + Vite
- **Деплой:** Render (бесплатный план)

## Запуск

### Сервер
```bash
cd server
npm install
npx prisma db push
npm run dev
```

### Клиент
```bash
cd client
npm install
npm run dev
```

## Деплой на Render
1. Загрузите код на GitHub
2. Подключите Render к репозиторию
3. Используйте `render.yaml` для автоматической настройки
4. Задайте переменные окружения (DATABASE_URL, JWT_SECRET)
